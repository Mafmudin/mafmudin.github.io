ProDialog

Library ini digunakan untuk membuat sebuah tampilan progres dialog dengan menggunakan gambar bergerak. Gambar bergerak yang dapat digunakan diantaranya SVG dan GIF. Dalam pembuatan UI/UX pada aplikasi android progress dialog adalah sebuah tampilan yang sangat dibutuhkan, dalam proses memuat data progress dialog akan ditampilkan sebelum data hasil muatan di tampilkan ke dalam view. Dengan adanya progress dialog maka view tidak akan menampilkan data jika data yang di muat belum siap. Jadi progress dialog adalah tampilan ketika proses pemuatan data berlangsung.

Andorid studio pada dasarnya sudah menyediakan progress dialog, akan tetapi untuk sesi ini saya membuat sebuah library yang dapat digunakan untuk menampilkan gambar bergerak pada progress dialog.
